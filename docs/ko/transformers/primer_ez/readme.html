Primer: Searching for Efficient Transformers for Language Modeling

다음은 Primer: Searching for Efficient Transformers for Language Modeling 논문의 PyTorch 구현입니다.

저자들은 트랜스포머 아키텍처를 위해 진화적인 탐색을 수행합니다. 그들은 이 탐색을 통해 찾은 아키텍처를 Primer (PRIMitives searched transformER)라고 명명합니다. Primer EZ 는 원래의 트랜스포머와 비교하여 Primer에서 가장 견고한 두 가지 수정 사항을 포함한 아키텍처입니다. Primer EZ는 일반적인 트랜스포머에 비해 훈련 속도가 훨씬 빠릅니다.

ReLU 제곱

탐색에서 찾은 가장 효과적인 수정은 position-wise feedforward module에서 ReLU 대신에 ReLU를 제곱하여 사용한 것입니다.

다중-DConv-헤드 어텐션 (MDHA)

다음 효과적인 수정은 쿼리, 키, 값들을 프로젝션한 후에 사용한 3 X 1 합성곱입니다. 이 합성곱은 시퀀스 차원과 (깊이별) 채널별로 이루어집니다. 다시 말해서, 각 헤드의 채널수가 d_k 라면, 합성곱은 각 d_k 채널별로 1 X 3 커널을 가질 것입니다.
